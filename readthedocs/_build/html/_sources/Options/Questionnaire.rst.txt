Questionnaire
=============

Le questionnaire prend les informations nécessaires sur le polygones sélectionné. Les renseignements suivant sont de mise:

* Type : Le type du polygone peut être une maison, une infrastructure, un cimétière,...
* Statut: Le statut détermine si c'est habité ou pas 
* Nombres de personnes : Le nombre de personne qu'on peut compter pour le polygone
